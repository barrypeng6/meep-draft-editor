React draft.js - Meep Draft Editor

此專案由 draft.js Facebook 開源專案發展而成，命名為 Meep Draft Editor 是一款對開發者自由度很高的一個文字編輯器開源專案，此為專案的 demo example 頁面，目前專案正在 發展中，如有任何問題請至專案觀看 Github README document 或發 pull request 給此專案幫助成長，也正在徵求共同貢獻者中。
